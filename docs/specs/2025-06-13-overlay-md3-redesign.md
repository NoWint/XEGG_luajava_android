# XEGG 悬浮窗 MD3 全面重构设计

## 目标
用 ComposeView 重写悬浮窗，复用主 App 的 XeggTheme + MD3 组件，覆盖 GG/RLGG 90% 核心功能。

## 架构
- OverlayService 使用 ComposeView 渲染 UI
- 复用 XeggTheme（深浅双主题 + MD3 色板）
- 6 个 Tab 页 + 悬浮球

## Tab 页功能

### 1. 搜索 Tab
- 搜索值输入框
- 数据类型：I8/I16/I32/I64/U8/U16/U32/U64/F32/F64/UTF8/Hex
- 搜索模式：精确 / 模糊 / 组合 / 范围
- 模糊条件：相等/不等/大于/小于/增加/减少
- 区域过滤：FilterChip 组（All/Ca/Cb/Cd/Ce/Cf/Ch）
- 操作：新搜索 / 继续搜索 / 重置
- 结果列表：地址+值+快捷操作（保存/编辑/锁定/跳转/指针搜索）

### 2. 已保存 Tab
- 保存列表管理
- 每项：地址+值+锁定开关+编辑
- 批量操作：全选/批量编辑/批量锁定
- 导入/导出列表
- 上下文菜单

### 3. 内存 Tab
- Hex 视图 + 文本视图切换
- 地址跳转
- 区域信息
- 内存编辑
- 书签

### 4. 脚本 Tab
- 脚本路径 + 文件选择
- 内联代码编辑
- 执行/停止
- 控制台输出
- 脚本历史

### 5. 加速 Tab
- 速度滑块 0.1x-10x
- 预设：0.5x/1x/2x/5x/10x
- 自定义速度输入
- 应用/取消

### 6. 设置 Tab
- 附加状态+断开
- 搜索设置
- UI设置（透明度/字体大小）
- 打开主界面
- 停止悬浮窗

## 交互
- 悬浮球：MD3 FAB 风格，可拖拽，点击展开
- 面板可拖拽移动
- 结果项长按上下文菜单
- 最小化回悬浮球

## 技术方案
- ComposeView 替代原生 View
- 共享 XeggTheme 主题
- 状态通过 XeggBridge 单例共享
